== Description ==

Woo Set Price Note – is the plugin that facilitate WooCommerce Shop-manager to display custom text/message/unit with product price.

Contribution link: https://github.com/shshanker/woo-set-price-note



### The Plugin Features:

* Price Note - enables WooCommerce shop manager to display custom note with product price.

* Unit - Custom note might be product price per unit ( $ 123.45/ piece ).

* Offer - Custom note might be festival/occasion offers ( $ 123.45 - Christmas Offer ).

* Stock Message - Custom note might be stock of products ( $ 123.45 | Limited Edition ). 

* Misc - Custom note might be whats on your mind.  



== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* Go to specific woo-commerce product page.
* Go to product data section and in general section you can find couple of extra fields.
* Set your message and set separator between price and the message.
* Update your product.
* Enjoy your message.
